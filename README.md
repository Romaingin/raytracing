# Raytracing Software
### ENSTA IN104

C++ parallelized implementation of a raytracing software.

## Libraries

GLM, SDL2 and OpenMP required.
The following packages thereby need to be present on your computer :

`libglm-dev`
`libsdl2-dev`
`libsdl2-image-dev`
`libsdl2-ttf-dev`
`libsdl2-gfx-dev`
`libopenmpi-dev`
`openmpi-bin` openmpi-doc`
